# Popcorn

Steps Demo App:
- Get [TMDB](https://www.themoviedb.org/) Api Key /settings / API / Request an API Key / Developer / Website and fill the fields
- Create React App
- Setup firebase hosting 
`npm i -g firebase-tools` 
- Get all the movies
- Build the rows
- Build the banner
- Build the Nav Bar
- Add trailer popups
